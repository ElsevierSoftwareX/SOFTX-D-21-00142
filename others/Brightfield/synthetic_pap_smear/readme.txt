- Download and install DipImage from http://www.diplib.org/
- Download the distmesh toolbox from http://persson.berkeley.edu/distmesh/. Put the library under General_functions/distmesh/
- Mex build General_functions/pnpoly.cpp for your platform.
- Start the simulation by running generate_synthetic_image.m.
- To change simulation parameters edit default_parameters.m.
- Main generation code is found in generation_script.m.
- For general information regarding how everything works refer to the paper "Simulation of Bright-Field Microscopy Images Depicting Pap-Smear Specimen" published in Cytometry Part A.
